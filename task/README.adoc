= MS SKF

== To start redis you need
[source%autofit,bash]
----
$ cd docker
$ docker-compose build
$ docker-compose up
----

== To start application you need
[source%autofit,bash]
----
$ ./mvn spring-boot:run
----

== Swagger
[source%autofit,bash]
----
$ Swagger url http://localhost:8080/skf/swagger-ui.html
----
